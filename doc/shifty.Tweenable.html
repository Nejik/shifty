Configure a tween that will start at some point in the future.

Parameters:
Name Type Description
config Object

See below
Properties:
Name Type Attributes Description
config.from Object <optional>

Starting position. If omitted, shifty.Tweenable#get is used.
config.to Object <optional>

Ending position.
config.duration number <optional>

How many milliseconds to animate for.
config.delay number <optional>

How many milliseconds to wait before starting the tween.
config.start function <optional>

Function to execute when the tween begins. Receives the state of the tween as the first parameter and attachment as the second parameter.
config.step function <optional>

Function to execute on every tick. Receives get as the first parameter, attachment as the second parameter, and the time elapsed since the start of the tween as the third. This function is not called on the final step of the animation.
config.easing Object.<(string|function()|string|function())> <optional>

Easing curve name(s) or function(s) to use for the tween.
config.attachment * <optional>

Cached value that is passed to the step/start functions.
config.promise function

Promise constructor for when you want to use Promise library or polyfill Promises in unsupported environments.
